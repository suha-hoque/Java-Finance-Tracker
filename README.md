# Java Finance Tracker

A Java-based desktop application that enables users to securely track personal income and expenses, manage financial records, and monitor spending using a local database and an intuitive graphical interface.

## Overview
This project is a personal finance tracking system built to strengthen skills in object-oriented programming, secure authentication, database integration, and desktop application development. The application allows users to create accounts, log transactions, and store financial data locally in a structured and secure manner.

## Features
- Secure user authentication with hashed credentials
- Add, edit, and delete income and expense transactions
- Expense categorization for better financial organization
- Persistent local storage using SQLite
- Desktop graphical user interface built with Java Swing
- Modular, object-oriented project structure

## Tech Stack
- Java
- Java Swing (GUI)
- SQLite
- JDBC
- Git & GitHub
